# Simple Weather App

This project is a simple weather app using vanilla Javascript and [OpenWeather API](https://openweathermap.org/).

## Technologies
- Javascript
- HTML5
- css
## How to run
1. (optional) install dev dependencies `npm install` or `yarn`
2. run the app opening `index.html` in the browser
